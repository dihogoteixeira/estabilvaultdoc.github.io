# System Backend

The system backend is a default backend in EstabilVault that is mounted at the `/sys`
endpoint. This endpoint cannot be disabled or moved, and is used to configure
EstabilVault and interact with many of Vault's internal features.

For more information about a particular path, please click on it in the sidebar.
