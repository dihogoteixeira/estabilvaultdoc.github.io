# EstabilVault Agent Auto-Auth CF Method

The `cf` method performs authentication against the [CF Auth
method](/docs/auth/cf).

## Credentials

The EstabilVault agent will use the `CF_INSTANCE_CERT` and `CF_INSTANCE_KEY` env variables to
construct a valid login call for CF.

## Configuration

- `role` `(string: required)` - The role to authenticate against on Vault.
